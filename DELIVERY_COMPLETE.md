## 💾 Data & Storage

**Location**: `Admin/shared/json/emergency-alerts.json`

**Format**: JSON array of alert objects

**Example Alert**:
```json
{
  "id": "alert_123456",
  "title": "Fire in Building A",
  "type": "fire",
  "location": "3rd Floor",
  "description": "Active fire detected",
  "severity": "critical",
  "instructions": "Evacuate immediately",
  "activeUntil": "2024-04-10T15:00:00Z",
  "createdAt": "2024-04-10T14:00:00Z",
  "status": "active"
}
```

**Features**:
- Auto-stores last 100 alerts
- Auto-removes expired alerts
- No cleanup needed
- Simple text format
